# cooksmart
<div align="center">
## CookSmart – Recipe & Drink Management System
</div>

CookSmart is a full-stack web application built using the MERN stack (MongoDB, Express.js, React, Node.js).
It provides a centralized platform for users to explore, manage, and organize both food recipes and drink preparations in a smart and efficient way.

## Frontend

The frontend is developed using React, offering a dynamic and responsive user interface.

Key Pages:
Home Page (Recipes & Drinks Listing)
Recipe Details Page
Add Recipe Page
Search & Filter Page
User Dashboard

The UI is designed using HTML, CSS, and JavaScript to ensure smooth navigation and a user-friendly experience.

API Communication:

The frontend uses Axios for:

Fetching recipes and drinks
Adding new recipes
Searching by ingredients
User interactions
##  Backend

The backend is built using Node.js and Express.js, handling all application logic.

It manages:
Recipe and drink data handling
API creation and routing
Request and response processing
Server-side validations
## Database

The application uses MongoDB Atlas for cloud-based data storage.

## Collections:
Users → Stores user details
Recipes → Stores food recipes
Drinks → Stores beverage data
Categories → Recipe classifications
 Features
Add, update, and delete recipes
Manage both food and drinks in one platform
Search recipes by ingredients
Browse recipes easily
Organized and centralized data storage
Responsive and clean UI
Fast API integration
 ## Tech Stack

Frontend: React
Backend: Node.js, Express.js
Database: MongoDB

## Installation and Setup

Clone the repository and install dependencies:

npm install
 Running the Application
Backend Server:
cd cooksmart
cd backend
npm run dev
Frontend Application:
cd cooksmart
npm run dev
 Frontend Dependencies

React Router DOM

npm install react-router-dom

Axios

npm install axios

Bootstrap / Tailwind CSS

npm install bootstrap
Backend Dependencies

Express.js

npm install express

Mongoose

npm install mongoose

Dotenv

npm install dotenv
 Authentication & Security

(Optional / If Used)

bcryptjs

npm install bcryptjs

jsonwebtoken (JWT)

npm install jsonwebtoken
